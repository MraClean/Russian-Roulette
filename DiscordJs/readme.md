> # Self Host
>
>Requirements:
>- node.js
>- discord bot
>- mongodb account
>- discordBots.org bot token
>
> Setup
>
> Use `npm i`
>
> Add an .env
>
>In the .env type:
>- token = discord_token
>- mongodb = mongodb_connection_string
>- dblToken = dbl_token
>
> Running
>
> Use node .
